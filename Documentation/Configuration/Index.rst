Extension configuration
=======================

The following extension configurations are provided through the settings module:

.. t3-field-list-table::
 :header-rows: 1

 - :Parameter:    Parameter
   :Description:  Description
   :Default:      Default

 - :Parameter:    configureFlux
   :Description:  If set disables the `flux` extension configuration parameters `autoload` and `plugAndPlay`.
   :Default:      1

 - :Parameter:    disableFluxPageLayoutSelector
   :Description:  If set disables the `Page Layouts`-tab in the page property form.
   :Default:      0
