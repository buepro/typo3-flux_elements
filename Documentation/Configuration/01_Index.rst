Extension configuration
=======================

The following extension configurations are provided through the settings module:

=============================== ========================================================================== ===========
Parameter                       Description                                                                Default
=============================== ========================================================================== ===========
configureFlux                   If set disables the `flux` extension configuration parameters `autoload`   1
                                and `plugAndPlay`.
disableFluxPageLayoutSelector   If set disables the `Page Layouts`-tab in the page property form.          1
=============================== ========================================================================== ===========
